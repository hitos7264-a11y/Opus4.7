1 分で読む Opus 4.7

What's New

4.6 → 4.7 の主要な変化

VISION
視覚解像度が約 3 倍に
長辺 1,568px → 2,576px、画素数で 3.25 倍。 XBOW では 54.5% → 98.5% に跳ね上がり、スクリーンショット読取りや座標ベース自動化が実用域に。
→ ビジョン詳細へ
REASONING
xhigh 追加 / Adaptive 一本化
effort に xhigh が新設され、Claude Code の既定が全プランで昇格。 一方で thinking.budget_tokens は廃止、temperature 系も固定化。
→ 推論の設計を読む
CONTEXT
長ctx retrieval は退行
総合 91.9% → 59.2%。524k〜1024k tokens 帯では 78.3% → 32.2%。 1M window を額面通りに使うのは危険。Anthropic は「以前の評価が甘かった」と説明。
→ コンテキスト詳細へ
SAFETY
リアルタイム Cyber Safeguards
サイバー高リスク用途の検出・遮断を自動化。正当な用途向けには Cyber Verification Program を同時立上げ。
honesty rate は 92%（Mythos 95.4%）。
→ 安全性 & Mythos を読む

過去最大級の横展開。Bedrock は次世代推論エンジンへ基盤刷新＋ゼロオペレータアクセス。 Copilot は全 IDE で Pro+/Business/Enterprise が順次対応。
