配信チャネル

単一のフラッグシップモデルとしては過去最大の広がりで同日ローンチ。 「ガバナンス込み」でエンタープライズ本番系を最初から想定した配信設計。

FIRST-PARTY
Anthropic Claude / API
モデル ID は claude-opus-4-7。Claude 公式アプリと Messages API、 Platform Docs 側から adaptive thinking などの新 API が記述されている。
AWS
Amazon Bedrock
US East (N. Virginia) / Asia Pacific (Tokyo) / Europe (Ireland) / Europe (Stockholm) の 4 リージョンで同日利用開始。単に新モデルをぶら下げたのではなく、 プロビジョニング不要で動的スケールする次世代推論エンジンに基盤刷新。 顧客のプロンプト/応答は Anthropic 側にも AWS 側にもオペレータが覗けない ゼロオペレータアクセスの設計。
GOOGLE
Vertex AI
Google Cloud Vertex AI Model Garden より同日利用開始。 Google Cloud の IAM / VPC Service Controls の下で運用可能。
MICROSOFT
Microsoft Foundry
AI Azure の Foundry Models カタログに同日追加。 Azure の既存セキュリティ/コンプライアンス境界を利用可能。
SNOWFLAKE
Snowflake Cortex AI（Public Preview）
米国・EU で Public Preview。Cortex Code（自然言語からのパイプライン生成）、 AI_COMPLETE 関数（構造化/非構造化データ分析）、 Cortex REST API（エージェンティックなエンタープライズアプリ構築） までが、Snowflake のセキュリティ境界の中で完結。
DEV TOOLS
GitHub Copilot
Pro+ / Business / Enterprise の各プランで、以下クライアントのモデルピッカーから順次選択可能：
Visual Studio Code、Visual Studio、Copilot CLI、GitHub Copilot Cloud Agent、 github.com、iOS / Android モバイル、JetBrains、Xcode、Eclipse。
プレミアムリクエスト倍率 7.5×（4/30 までプロモ）。 Opus 4.5 / 4.6 の選択肢は数週間以内に廃止予定。

OpenRouter の API Pricing & Providers にリスティング。 サードパーティの集約経路からも同様のレートで呼び出せる。
