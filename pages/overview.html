「天井をあえて抑える」という思想

Anthropic は 2026 年 4 月初頭から、Claude Mythos Preview をごく一部の選ばれた企業だけに解放する動きを始めていた。 Project Glasswing という取り組みのなかで動いていたこの Mythos は、 攻撃再現性を測る CyberGym で 83.1%（4.6 は 66.6%）、 SWE-bench Verified で 93.9% という水準。

Anthropic はこれを広く放つ決断をまだ下しておらず、代わりに「Mythos 級モデルを 将来どう安全に展開するか」を学ぶための 露払い として、 能力を少し絞ってセーフガードを実戦投入した形で用意したのが Opus 4.7。 CNBC が報じた Dario Amodei の説明では、4.7 は 「Mythos ほど包括的に強くはないが、生産環境で安心して任せられる、 現時点で一般公開されているもっとも有能な Claude」という役回り。

この仕様書の背景理解
ベンチマークの「最強値」ではなく、安全に本番投入できるもっとも強いモデル という基準に競争軸が移っている、という前提で 4.7 の仕様は読む必要がある。

Opus 4.7 が持ち込んだ変化

$5 / $25 per MTok は不変。新トークナイザで 1.00〜1.35× のインフレが発生。 コードや構造化データ、非英語テキストでは特に膨張率が大きい。
low / medium / high / xhigh / max の五段階。 Claude Code の既定は全プランで xhigh に昇格。 一方で budget_tokens は廃止、temperature 等は固定化。
長辺 1,568px → 2,576px、画素数ベースで約 3.25 倍。 XBOW は 54.5% → 98.5%、社内コーディング 93 タスクは +13pt。
総合 91.9% → 59.2%。524k〜1024k 帯では 78.3% → 32.2%。 Anthropic は「以前の評価が甘かった」という立場だが、現場では賛否。
サイバー高リスク用途の自動遮断を導入。Cyber Verification Program で 正当な研究者には解除する運用。
