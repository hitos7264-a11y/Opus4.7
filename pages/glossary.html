用語集

本仕様書で頻出する用語・略語の定義。

Opus 4.7 で導入された、思考配分をモデル側で最適化する仕組み。 ユーザーは effort だけを指定し、budget_tokens のような細かい指定は不可。
思考の強さを決める単一のダイヤル。low / medium / high / xhigh / max の 5 段階。
4.7 で新設された effort 段階。high と max の中間で、長時間エージェント・ Claude Code の既定。探索的ツール呼び出しを積極化。
Mythos は限定プレビューで動いている高能力 Claude。 Project Glasswing は重要ソフトウェアの保護を目的とする Anthropic の取り組みで、 Mythos はその中核 AI。
Anthropic が公表するモデルの詳細評価ドキュメント。Opus 4.7 のものは 232 ページ。 アライメント評価、安全性、既知のリスク、訓練時の事故などを開示。
サイバー高リスク用途をリアルタイムに検出・遮断する機構。4.7 で実戦投入。
正当な用途（脆弱性研究、ペネトレ、レッドチーム）向けに、 Cyber Safeguards の対象カテゴリーを解放するための審査登録制度。
攻撃再現性を測るベンチマーク。4.6 が 66.6%、Mythos が 83.1%。4.7 は非公開。
ソフトウェアエンジニアリング能力を測る代表的ベンチ。 Verified は正解が厳密検証された版、Pro はより難度が高く多言語を含む拡張版。
Anthropic 社内の視覚精度ベンチ。小さな文字や細密図形の読取り課題で、 4.7 は 98.5%、4.6 は 54.5%。
ターミナル操作を含むエージェント評価。4.7: 69.4%、GPT-5.4: 75.1%。
コンピュータ操作を総合評価する OSWorld の検証済み版。4.7: 78.0%。
Model Context Protocol（MCP）を使ったエージェント運用を測るベンチ。 4.7: 77.3%。
「知識労働」型評価。Elo スコアで測定。4.7: 1753、GPT-5.4: 1674、Gemini 3.1 Pro: 1314。
ウェブリサーチ型のベンチ。4.7: 79.3%、GPT-5.4 Pro: 89.3%、Gemini 3.1 Pro: 85.9%。
主題横断的な一般化能力を測るベンチ。high effort で 80.6 → 72.8 と退行したことが r/singularity で話題に。
Artificial Analysis が公表する複合スコア。Opus 4.7 (max) は 57。
金融業務エージェントの評価。Anthropic が社内で重視。4.7: 64.4%。
法務系長文書面レビューを測るベンチ。4.7 high effort で 90.9%。
Anthropic の価格割引機構。キャッシュで入力最大 90% OFF、 バッチ処理でさらに 50% 割引を積める。
