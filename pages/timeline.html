タイムライン

Opus 4 系列および Mythos / Glasswing 周辺の主要イベントを時系列で。

2026-02
Claude Opus 4.6 公開
Opus 4 系列の前世代。CyberGym 66.6% / SWE-bench Verified 80.8% / 長 ctx retrieval 総合 91.9% など、4.7 の比較基準となる数字群がここで確定。
2026-04 初頭
Project Glasswing / Mythos Preview 始動
重要ソフトウェア保護をテーマとする Project Glasswing が公表され、 Claude Mythos Preview が一部企業に限定解放。CyberGym 83.1% / SWE-bench Verified 93.9% という段違いの能力が一部にリークし、業界を揺るがす。
2026-04-16（木）
Claude Opus 4.7 一般公開
Anthropic が Mythos を広く放つ代わりに、能力を意図的に絞り、 セーフガードを実戦投入した Opus 4.7 を一般公開。Amazon Bedrock、 Google Cloud Vertex AI、Microsoft Foundry、Snowflake Cortex AI、 GitHub Copilot などで同日ローンチ。
2026-04-16
報道各社が一斉配信
CNBC / VentureBeat / Help Net Security / Interesting Engineering / Mashable / Economic Times などが同日報道。VentureBeat は「僅差で王座奪還」、 Mashable は「PR か実質かで意見割れる」と温度差のある見出し。
2026-04-17
コミュニティで回帰報告が噴出
r/ClaudeAI では「長 ctx retrieval の regression」「50% 実請求増」などの 批判スレッドが並ぶ。Hacker News ではシステムカードの Mythos 言及回数が 熱心に読み込まれ、メタな議論が展開される。
2026-04-18
本仕様書の執筆時点
Caylent / Finout / APIYI などから移行ガイド・コスト解析が出揃い、 FinOps 視点と実装視点の両方から「Drop-in 置換は危険」という合意が形成される。 GitHub Copilot のプロモ期限（4/30）までに 4.5/4.6 からの移行を完了させる動きが活発化。

Mythos の一般公開時期、4/30 以降の Copilot プレミアム倍率、 長 ctx retrieval のパッチ更新、Opus 4.5 / 4.6 の deprecation 日程、 GDPVal・Finance Agent での競合との再対戦など。
